# Ahmed ElGabbas Portfolio

## Overview

A modern and responsive personal portfolio website built using React, Vite, Tailwind CSS, and Framer Motion. The portfolio showcases my skills, projects, certifications, leadership experience, and contact information in a professional and interactive design.

## Features

* Responsive design for all devices
* Modern UI with smooth animations
* Interactive skills section
* Certificates showcase with previews
* Project portfolio section
* Downloadable CV
* Contact form integration using Formspree
* Optimized performance with Vite
